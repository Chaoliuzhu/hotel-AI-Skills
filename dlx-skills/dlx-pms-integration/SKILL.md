# dlx-pms-integration — 德胧定制PMS系统集成

> 支持德胧旗下酒店与主流PMS系统的数据对接，实现运营自动化。

## 支持的PMS系统

| 系统 | 厂商 | 接口类型 | 支持程度 |
|------|------|----------|----------|
| **FOSS** | 石基 | REST API | ⭐⭐⭐ 完整支持 |
| **Opera (Oracle)** | Oracle | OPERA Web Services | ⭐⭐ 部分支持 |
| **西软** | 西软 | 文件/数据库 | ⭐⭐ 部分支持 |
| **别样红** | 别样红 | REST API | ⭐⭐⭐ 完整支持 |
| **绿云** | 绿云 | REST API | ⭐⭐⭐ 完整支持 |
| **住哲** | 住哲 | REST API | ⭐⭐ 完整支持 |

## 核心场景

### 1. 房态实时同步
- **当前房态**：查询各房型可用房间数
- **未来房态**：未来N天的预测可用数
- **维修房**：标记清理中/维修中的房间
- **房态变化**：当房态变更时自动推送通知

### 2. 预订查询与管理
- **按预订号查询**：获取完整预订详情
- **按客人姓名查询**：查找历史预订记录
- **按日期范围查询**：某时段的所有预订
- **预订变更**：接收来自OTA/直销渠道的预订变更

### 3. 入住/退房处理
- **预到查询**：查询明日预期入住名单
- **快速入住**：获取客户信息+房间分配+房卡准备
- **退房处理**：账单结算+发票开具+客户离店状态
- **延迟退房**：延长住宿的处理

### 4. 账单与账务
- **账单查询**：查询客人账户的消费明细
- **账务推送**：将额外消费（餐/吧/其他）记账到房账
- **账单结算**：处理付款方式（现/卡/转账）
- **账务核对**：每日应收账款核对

### 5. 客户信息
- **客史查询**：历史入住记录、消费偏好
- **VIP识别**：自动识别VIP客户触发服务预警
- **黑名单检查**：客人风控检查

## 安全与权限

### 认证方式
- **API Key**：每个酒店分配独立Key
- **签名验证**：请求参数签名防篡改
- **IP白名单**：仅允许德胧服务器IP调用

### 数据安全
- 客户敏感信息（身份证/手机号）需加密存储
- 对外展示时自动脱敏（只显示后4位）
- 访问日志完整留存

## 错误处理

| 错误码 | 含义 | 处理建议 |
|--------|------|----------|
| E001 | 系统维护中 | 稍后重试，或使用备用系统 |
| E002 | 酒店编号不存在 | 检查酒店编号是否正确 |
| E003 | 接口调用超限 | 降低调用频率 |
| E004 | 认证失败 | 检查API Key是否有效 |
| E005 | 房型不可用 | 推荐相近房型或候补 |
