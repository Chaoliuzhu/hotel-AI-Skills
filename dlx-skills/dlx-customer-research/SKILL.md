## 四大核心客群画像体系

本 skill 围绕德胧酒店四大核心客群展开系统性调研，每类客群均覆盖：

### 客群一：商旅客群

> 核心标签：出差 · 会议 · 差旅报销 · 效率优先

**典型画像**
- 身份：企业白领、中层管理者、销售/BD、异地办公人员
- 年龄：28-50岁
- 出差频率：每月1-4次
- 报销方式：公司差旅报销（关注发票合规性）
- 决策者：本人（自选）或行政/财务代订

**高频场景**
1. **日常出差** — 单晚住宿，第二天退房继续行程
2. **客户拜访配套** — 约客户在酒店餐厅见面，兼顾私密与便利
3. **跨城会议** — 半天或全天会议，使用会议室+客房套餐
4. **当日往返** — 早晚高铁/开车，中午抵离，不过夜

**消费动机（优先级排序）**
1. 位置便利 — 距客户/高铁站/机场近是首选
2. 早餐品质 — 出差在外早餐往往是唯一正餐
3. 商务配套 — 稳定Wi-Fi、充电接口、写字台
4. 发票规范 — 专票普票清晰，可用于报销
5. 安静程度 — 商务人士对噪音敏感

**决策因素**
- 差标范围内优先（公司报销上限）
- 协议价格（与酒店签有协议价的企业客户）
- 评价口碑（同事推荐/携程评分）
- 地理位置（距客户或交通枢纽的便捷性）

**触达渠道**
- 企业协议签订（BD团队直销）
- 携程/美团企业版
- 差旅管理平台（携程企旅、Trip.com）
- 商务型内容投放（LinkedIn、行业媒体）

**沟通话术要点**
- 强调"效率感"：快速入住/退房、叫醒服务、商务套餐
- 突出"合规报销"：发票规范、专票速开
- 避免过于休闲的营销语言（度假感话术对商旅客人适得其反）

### 客群二：度假客群

> 核心标签：周末游 · 亲子 · 滨海度假 · 家庭出行

**典型画像**
- 身份：有孩家庭（孩子3-12岁为主）、情侣、闺蜜
- 年龄：30-45岁（带孩子）/ 25-35岁（情侣/闺蜜）
- 出游类型：周末短途游、小长假周边游、暑假滨海季
- 出行方式：自驾为主（京津冀地区）
- 决策者：通常是女性（妈妈主导预订决策）

**高频场景**
1. **周末亲子游** — 周六上午出发，周日午后退房，孩子玩酒店设施
2. **滨海亲海季** — 夏季海边玩水、东疆亲海公园打卡
3. **节假日度假** — 五一、国庆小长假，连住2-3晚
4. **Staycation** — 不去景点，就在酒店"宅着"放松
5. **特殊节点** — 生日纪念日、情人节、母亲节

**消费动机（优先级排序）**
1. 亲子友好设施 — 儿童泳池、亲子房、托管服务
2. 滨海景观 — 海河景观、落地窗、能看到海的房间
3. 拍照出片 — 朋友圈打卡需求，视觉呈现重要
4. 餐饮体验 — 早餐、海鲜自助、下午茶
5. 周边可玩性 — 距景点/海滩/美食街的动线

**决策因素**
- 社交媒体口碑（小红书、抖音、朋友圈）
- 亲子友好程度（有无儿童设施、亲子房）
- 评价中的真实入住体验（图片是否与实际相符）
- 性价比（节假日溢价接受度较高）
- 天气/季节因素

**触达渠道**
- 小红书（图文/视频种草）
- 抖音/视频号（短视频、探店内容）
- 朋友圈广告
- 美团/携程亲子/度假标签筛选
- 亲子社群、KOL推荐

**沟通话术要点**
- 营造"温暖回忆感"：家庭时光、陪伴、仪式感
- 画面感描述：落地窗海景、儿童泳池、精致早餐托盘
- 强调"出片"和"打卡"价值
- 善用时令节点：暑期玩海、清明踏青、圣诞氛围

### 客群三：本地散客

> 核心标签：周末聚餐 · 本地Staycation · 餐饮消费者 · 会员复购

**典型画像**
- 身份：塘沽/滨海本地居民、高净值回头客
- 年龄：35-55岁
- 消费场景：不外宿，仅消费餐饮/婚宴/SPA等单项目
- 决策者：本人自主决策

**高频场景**
1. **周末家庭聚餐** — 家庭聚会、老人寿宴、节假日团圆饭
2. **婚宴/宝宝宴** — 重要仪式场合，桌数10-40桌
3. **本地Staycation** — 不出城，在酒店住一晚换环境放松
4. **下午茶/商务茶歇** — 闺蜜下午茶、商务接待茶歇
5. **健身/SPA** — 健身房、游泳池消费

**消费动机（优先级排序）**
1. 品质感 — 环境、口味、服务均需体面
2. 便利性 — 距家近，无需长途开车
3. 私密性 — 包间、家庭聚会场景
4. 性价比 — 本地客对价格敏感度高于游客
5. 会员权益 — 积分抵扣、会员折扣

**决策因素**
- 曾在酒店住宿或用餐的正向体验
- 朋友/亲人口碑推荐
- 会员积分/权益体系
- 促销活动（团购套餐、节日折扣）
- 餐饮点评（大众点评、美团）

**触达渠道**
- 酒店会员体系（App/公众号推送）
- 大众点评/美团店铺页
- 朋友圈本地广告
- 老客户推荐（口碑转介绍）
- 物业/街道合作（社区活动）

**沟通话术要点**
- 强调"熟悉感"和"品质稳定"
- 突出会员专属权益和积分价值
- 餐饮类内容以"家宴感"为核心
- 婚宴/宝宝宴强调仪式感和专属定制

### 客群四：企业客户

> 核心标签：协议客户 · 会议团建 · MICE · 年度协议

**典型画像**
- 身份：企业HR、行政采购、活动策划负责人
- 企业规模：50-500人中小型企业为主
- 行业：制造业、贸易、科技、金融（滨海新区优势行业）
- 决策链：HR/行政提出方案 → 财务审批 → 管理层拍板

**高频场景**
1. **协议住宿** — 员工出差、异地办公驻店，签协议价
2. **会议场地** — 10-100人会议，使用多功能厅
3. **团队团建** — 年会、季度团建，半天或全天包场
4. **培训/拓展** — 企业内训、拓展训练，住宿+场地套餐
5. **客户招待** — 重要客户来访，住宿+宴请一条龙

**消费动机（优先级排序）**
1. 协议价格 — 低于前台价，有专属客户经理
2. 场地配套 — 多功能厅、投影、音响、茶歇
3. 餐饮承接力 — 大型宴席（10-40桌），婚宴/年会经验丰富
4. 地理位置 — 滨海新区核心位置，距客户近
5. 服务响应速度 — 专属客户经理，快速确认需求

**决策因素**
- 协议价格竞争力
- 场地档期可配合性
- 酒店品牌背书（开元名都品牌效应）
- 往期活动经验（案例展示）
- 账期和结算方式（对公转账、月结）

**触达渠道**
- BD直销（酒店销售团队主动拜访）
- 行业展会（滨海新区制造业展会）
- 携程企旅企业版
- 老客户推荐（口碑介绍新客户）
- 政府/园区合作（滨海新区招商局合作酒店）

**沟通话术要点**
- 专业、严谨、有案例支撑
- 强调"一站式解决方案"：住宿+会议+餐饮全包
- 提供场地平面图、过往活动照片等真实素材
- 可提供定制化方案（按企业需求灵活组合）

## 调研方法论

### 线上调研（数字水源法）

| 客群 | 优先平台 | 提取内容 |
|------|----------|----------|
| 商旅客群 | 携程商旅社区、LinkedIn、脉脉 | 出差选酒店标准、差旅痛点 |
| 度假客群 | 小红书、抖音、微博旅行 | 种草/拔草内容、决策因素 |
| 本地散客 | 大众点评、美团、微信朋友圈 | 餐饮评价、聚会场景口碑 |
| 企业客户 | 慧聪网、阿里巴巴商业社区 | MICE采购、协议客户案例 |

### 线下调研（直接访谈）

- **访谈对象**：前台接待、销售部、客房部、餐饮部（直接接触客人的一线员工）
- **访谈问题示例**：
  - "最近一个月，哪类客人来得多？"
  - "客人最常问的问题是什么？"
  - "客人投诉最多的是什么？"
  - "什么情况下客人会成为回头客？"
- **问卷设计**：针对四大客群分别设计3-5题专项问卷，通过公众号/短信/二维码发放

## 容错机制

### 异常处理

- **客群类型不在四大核心客群内（如"会议型客户"）**：先映射到最近的已有客群（如"会议型"→"企业客户"子场景）；标注「已映射至企业客户中的会议场景」
- **特定平台数据获取失败（如小红书API不可用）**：切换备选平台；小红书失败→改用抖音/微信搜索；所有平台均失败→基于已有知识库输出，标注「⚠️ 部分数据未能实时获取，基于历史数据补充」
- **竞品对比数据缺失**：使用公开可得信息（OTA评分、价格区间）补充；无数据时标注「[竞品名]：数据待补充，建议线下调研」
- **调研报告Token超出限制**：自动压缩为摘要版（保留四大客群核心结论，细节折叠）；询问用户是否需要展开某一特定客群
- **缺少线下访谈数据（仅能做线上调研）**：明确标注「本报告基于线上公开数据，线下验证建议补充」；提供访谈问题清单供用户自行执行

### 决策边界

| 情况 | 决策 |
|------|------|
| 用户要求的调研范围超出四大客群 | 聚焦四大客群，超出部分标注「建议单独立项」 |
| 数据来源质量存疑 | 标注数据置信度（高/中/低），不删除原始引用 |
| 用户要求极短时间内完成全量调研 | 降级为"快速概览"模式，输出精简版框架 |
| 某客群数据完全空白 | 输出该客群框架+「[此客群数据待补充]」，不阻断其他客群 |

### 检查点机制

每完成一个客群画像输出状态快照：`✅ 客群[X/4] 完成 | 商旅客群：核心标签已提取 | 高频场景：3个 | 触达渠道：4个`
