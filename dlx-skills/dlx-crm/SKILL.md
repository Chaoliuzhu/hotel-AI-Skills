# dlx-crm — 德胧定制客户关系管理

> 专为德胧集团旗下酒店构建完整的客户生命周期管理体系。

## 核心框架：四象限客户分类

| 象限 | 特征 | 策略 |
|------|------|------|
| **黄金客户** | 高消费 × 高忠诚度 | 重点维护、专属权益、优先服务 |
| **潜力客户** | 高消费 × 低忠诚度 | 提升体验、个性化关怀、积分激励 |
| **普通客户** | 低消费 × 高忠诚度 | 成本优化、适度激励、交叉销售 |
| **低价值客户** | 低消费 × 低忠诚度 | 控制成本、基础服务、偶尔优惠 |

## 会员体系设计

### 等级设计（参考开元会）
```
普通会员 → 银卡会员 → 金卡会员 → 钻卡会员
  0        5000      20000      80000
（单位：年度消费积分）
```

### 权益体系
- **积分兑换**：住房积分兑换、餐券、增值服务
- **会员专享价**：官网/直销渠道专属折扣（银卡9.5折/金卡9折/钻卡8.5折）
- **优先权益**：优先排房、延迟退房、免费加床
- **专属礼遇**：生日礼包、周年纪念、节日礼品

## VIP客户管理

### VIP分级
- **VVIP**：年度消费10万+，年度入住15晚以上
- **VIP**：年度消费5万+，年度入住8晚以上
- **商务VIP**：企业协议客户关键决策人

### 维护机制
```
VVIP：专属客户经理、每月一次主动联系、定制化服务
VIP：专属客服、季度关怀、重要节日祝福
企业VIP：企业关系维护、协议回顾、B端对接人维护
```

### 客史档案
- 基本信息：姓名、电话、企业、偏好（床型/楼层/房间设施）
- 消费记录：历史入住、房型偏好、消费金额
- 服务印记：特殊需求、投诉记录、表扬记录
- 价值评估：年度贡献值、忠诚度评分、潜力评级

## 私域运营

### 企微私域
- **拉新**：线下扫码（前台/餐厅/宴会厅）、线上预约
- **分层**：按消费/入住频次/偏好分层运营
- **触达**：每周优惠（周三会员日）、活动预告、节日祝福

### 朋友圈/社群
- 内容策略：酒店场景、生活方式、限时优惠、用户好评
- 推送节奏：每周2-3条，重要节点提前预热

## 流失预警与挽回

### 流失预警信号
- 入住频次下降（同比下降30%以上）
- 消费金额下降（同比下降40%以上）
- 差评/投诉后无后续
- 会员等级下降

### 挽回策略
- 定向优惠券（吸引回访）
- 专属客服回访（了解原因）
- 限时特权（到期提醒）

## 触发词与执行

当用户说以下内容时激活本技能：
- "查看这个客户的画像"
- "本周需要回访哪些VIP"
- "设计一个会员拉新活动"
- "分析客户流失原因"
- "如何提升复购率"
- "整理本月客户投诉"
- "给这位客人准备惊喜"
